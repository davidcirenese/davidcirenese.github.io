Weather Station

Powered by an ESP8266 microcontroller, I developed a device capable of communicating with a federal weather database via a REST API. The device also utilizes a BME280 to calculate relative humidity and temperature of the room. Additional features were added including being able to set an alarm, timer, or simply view the current time.
